# Assembler

## Running
```bash
python assembler.py <path to assembly file> <path to output file>
```

## Adding/Updating Ops 
Modify `ops.csv` to add or change any operations.

## Running Tests
To run the tests, from a bash prompt (e.g. Git BASH) run
```bash
./tests/run.sh
```
